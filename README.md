# Habit Tracker PWA

A cute, offline-first **Habit Tracker Progressive Web App (PWA)** with a weekly heart-check grid, daily progress, streak counter, mood emojis, notes, and monthly analytics.

Designed to feel like a real app on **iPad and Android**, with a calm Japanese/Korean-inspired kawaii UI and zero backend dependency.
